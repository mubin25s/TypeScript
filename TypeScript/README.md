# Simple TypeScript Problems

This directory contains basic TypeScript problems to help you get started with the syntax and logic.

## List of Problems:

1.  **hello_world.ts**: Basic console output.
2.  **sum_two_numbers.ts**: Addition logic and function usage.
3.  **even_or_odd.ts**: Conditional statements.
4.  **max_of_three.ts**: Using Math helpers and multiple parameters.
5.  **print_1_to_10.ts**: Loop structures.
6.  **area_of_circle.ts**: Mathematical calculations and constants.
7.  **celsius_to_fahrenheit.ts**: Formula implementation.
8.  **factorial.ts**: Loop-based factorial calculation.
9.  **simple_interest.ts**: Working with multiple numeric arguments.
10. **reverse_string.ts**: Basic string and array manipulation.

## How to run:

You can run these files using `ts-node` or by compiling them with `tsc` and running with `node`.

```bash
npx ts-node <filename>.ts
```
